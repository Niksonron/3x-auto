# 3x-ui VPN Automation (Русская версия)

Автоматизированные скрипты развертывания каскадной VPN инфраструктуры на VLESS + 3x-ui.

## Обзор

Проект предоставляет bash-скрипты для автоматического развертывания двухсерверной каскадной VPN инфраструктуры:

```
Клиент → RU relay сервер → Зарубежный VPS → WARP/Direct → Интернет
```

## Основные возможности

- **Автоматическое развертывание** RU relay сервера и зарубежного VPS
- **Защита и харденинг**: только SSH по ключу, UFW, fail2ban
- **VLESS поверх TCP с маскировкой Reality**
- **Два режима выхода**: прямой и через Cloudflare WARP
- **Комбинированная маршрутизация**: `RU → direct`, `default → proxy` (клиент + сервер)
- **Панель управления 3x-ui** только на зарубежном сервере
- **Автогенерация клиентских конфигураций** (.txt файлы)
- **Идемпотентные скрипты**: безопасны для повторного запуска
- **Сценарии**: health-check, update, uninstall

## Поддерживаемые системы

- Ubuntu 20.04+ / 22.04+
- Debian 11+ / 12+

## Быстрый старт

1. Клонируйте репозиторий
2. Скопируйте `.env.example` в `.env` и настройте параметры
3. Запустите установку:
   ```bash
   ./install.sh --all
   ```
   Или раздельно:
   ```bash
   ./install.sh --relay    # RU relay сервер
   ./install.sh --foreign  # Зарубежный VPS с 3x-ui
   ```

## Конфигурация

Все параметры задаются в файле `.env`.
